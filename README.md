# SMM সার্ভিস প্যানেল প্রজেক্ট

## প্রজেক্ট পরিচিতি

এই প্রজেক্টটি একটি সোশ্যাল মিডিয়া মার্কেটিং (SMM) সার্ভিস প্যানেল ওয়েবসাইট, যা [smmgen.com](https://smmgen.com/) এবং [demo.perfectpanel.com](https://demo.perfectpanel.com/) এর অনুপ্রেরণায় তৈরি করা হয়েছে। এই প্লাটফর্মটি দুটি মূল কম্পোনেন্ট নিয়ে গঠিত: একটি **অ্যাডমিন প্যানেল** এবং একটি **কাস্টমার প্যানেল**।

প্রজেক্টটি Next.js, React, Tailwind CSS, Prisma, PostgreSQL, এবং NextAuth দিয়ে তৈরি করা হয়েছে।

## বর্তমান অবস্থা

নিম্নলিখিত সেকশন অনুযায়ী প্রজেক্টের বর্তমান অবস্থা:

### 1. অ্যাডমিন প্যানেল (70% সম্পন্ন)

#### ✅ সম্পন্ন হয়েছে:
- ড্যাশবোর্ড: মূল মেট্রিক্স দেখানো (সর্বমোট ইউজার, অর্ডার, রাজস্ব)
- সার্ভিসেস: সমস্ত সার্ভিস তালিকা ও সম্পাদনা
- ক্যাটাগরি ম্যানেজমেন্ট
- অর্ডার ম্যানেজমেন্ট
- ইউজার ম্যানেজমেন্ট (বেসিক)

#### ❌ অসম্পূর্ণ কাজ:
- ফান্ড ম্যানেজমেন্ট
- সাপোর্ট টিকেট (এআই এবং মানব)
- চাইল্ড প্যানেল
- ঘোষণা সেকশন
- অ্যাফিলিয়েট সিস্টেম
- সেটিংস (সাইট কনফিগারেশন, অ্যাপিয়ারেন্স)
- ইন্টিগ্রেশন (লাইভ চ্যাট, অ্যানালিটিক্স, নোটিফিকেশন)

### 2. কাস্টমার প্যানেল (50% সম্পন্ন)

#### ✅ সম্পন্ন হয়েছে:
- ড্যাশবোর্ড: ব্যবহারকারীর মেট্রিকস
- নতুন অর্ডার প্লেসমেন্ট
- সার্ভিস ব্রাউজিং
- প্রোফাইল ম্যানেজমেন্ট

#### ❌ অসম্পূর্ণ কাজ:
- ম্যাস অর্ডার
- অর্ডার ম্যানেজমেন্ট (রিফিল, ক্যানসেল)
- ফান্ড অ্যাড করা
- ফান্ড লগ
- সাপোর্ট ইনবক্স
- অ্যাফিলিয়েট সিস্টেম
- API অ্যাকসেস

### 3. ফ্রন্টএন্ড (90% সম্পন্ন)

#### ✅ সম্পন্ন হয়েছে:
- রেসপনসিভ ডিজাইন
- হোমপেজ সেকশন (হিরো, সার্ভিসেস, টেস্টিমোনিয়াল)
- অথেনটিকেশন (লগইন, রেজিস্ট্রেশন)
- পেজ নেভিগেশন
- কম্পোনেন্ট স্ট্রাকচার

#### ❌ অসম্পূর্ণ কাজ:
- মাল্টি-ল্যাঙ্গুয়েজ সাপোর্ট
- কিছু কাস্টম পেজ (FAQs, টার্মস)

### 4. ব্যাকএন্ড (65% সম্পন্ন)

#### ✅ সম্পন্ন হয়েছে:
- ইউজার অথেনটিকেশন এবং অথরাইজেশন
- সার্ভিসেস এবং ক্যাটাগরি API
- বেসিক অর্ডার প্রসেসিং

#### ❌ অসম্পূর্ণ কাজ:
- পেমেন্ট গেটওয়ে ইন্টিগ্রেশন (আংশিক)
- AI টিকেট প্রসেসিং
- থার্ড পার্টি ইন্টিগ্রেশন
- অ্যাফিলিয়েট সিস্টেম API

## প্রতিটি সেকশন কী কী করতে হবে

### 1. অ্যাডমিন প্যানেল

#### ডাটাবোর্ড
- অ্যাড করতে হবে: রিয়েল-টাইম ড্যাশবোর্ড আপডেট, আরো ভিজুয়াল চার্ট

#### ইউজারস
- শেষ করতে হবে: ইউজার সাসপেন্ড করা, ইউজার ডিলিট করা, ফিল্টার এবং সার্চ

#### সার্ভিসেস
- আপডেট করতে হবে: বাল্ক সার্ভিস মডিফিকেশন, সিঙ্ক লগ ফিচার

#### অর্ডারস
- আপডেট করতে হবে: অর্ডার স্টেটাস ফিল্টারিং, রিফিল এবং ক্যানসেল অপশন

#### সাপোর্ট টিকেট
- সম্পূর্ণ বাকি: AI সাপোর্ট, হিউম্যান সাপোর্ট, কনডিশনাল লজিক

#### ফান্ডস
- সম্পূর্ণ বাকি: ট্রানজাকশন হিস্টরি, ইউজার ফান্ড অ্যাড, প্রাইস আপডেট

#### চাইল্ড প্যানেল
- সম্পূর্ণ বাকি: রিসেলার প্যানেল কনফিগারেশন

#### সেটিংস
- সম্পূর্ণ বাকি: জেনারেল সেটিংস, অ্যাপিয়ারেন্স, পেজেস, SEO, মেনু, ল্যাঙ্গুয়েজ, ইমেইল সেটাপ, ইন্টিগ্রেশন

### 2. কাস্টমার প্যানেল

#### অর্ডার
- আপডেট করতে হবে: ম্যাস অর্ডার, অর্ডার স্টেটাস ম্যানেজমেন্ট

#### ফান্ডস
- সম্পূর্ণ বাকি: পেমেন্ট গেটওয়ে ইন্টিগ্রেশন, ফান্ড লগ

#### সাপোর্ট ইনবক্স
- সম্পূর্ণ বাকি: নতুন টিকেট তৈরি, স্ট্যাটাস ট্র্যাকিং

#### API
- সম্পূর্ণ বাকি: API ডকুমেন্টেশন, কি জেনারেশন, অ্যাকসেস কন্ট্রোল

## API ডেভেলপমেন্ট প্রম্পট

### অ্যাডমিন প্যানেলের জন্য API এন্ডপয়েন্টস

```prompt
Next.js API route তৈরি করুন যা SMM সার্ভিস প্যানেলের অ্যাডমিন ফাংশনালিটি হ্যান্ডেল করতে পারে। নিম্নলিখিত ফিচারগুলি অন্তর্ভুক্ত করুন:

1. ড্যাশবোর্ড স্ট্যাটিসটিক্স API (মোট ইউজার, অর্ডার, রেভিনিউ ইত্যাদি পাওয়ার জন্য)
2. ইউজার ম্যানেজমেন্ট API (সমস্ত ইউজার পাওয়া, ইউজার আপডেট করা, সাসপেন্ড করা, ডিলিট করা)
3. সার্ভিস ম্যানেজমেন্ট API (সার্ভিস তৈরি, আপডেট, ডিলিট, বাল্ক আপডেট)
4. অর্ডার ম্যানেজমেন্ট API (অর্ডার স্ট্যাটাস আপডেট, রিফিল, ক্যানসেল)
5. টিকেট ম্যানেজমেন্ট API (AI এবং হিউম্যান টিকেট প্রসেসিং)

প্রতিটি এন্ডপয়েন্টে অ্যাডমিন অথেন্টিকেশন মিডলওয়ার ব্যবহার করুন এবং প্রয়োজনীয় ভ্যালিডেশন লজিক যোগ করুন।
```

### অ্যাডমিন ড্যাশবোর্ড কম্পোনেন্টস

```prompt
React এবং Tailwind CSS ব্যবহার করে SMM সার্ভিস প্যানেলের অ্যাডমিন ড্যাশবোর্ডের জন্য UI কম্পোনেন্ট তৈরি করুন। নিম্নলিখিত কম্পোনেন্টগুলি অন্তর্ভুক্ত করুন:

1. ড্যাশবোর্ড ওভারভিউ কার্ড (কি মেট্রিক্সের জন্য)
2. ড্যাটা ভিজুয়ালাইজেশন চার্ট (লাইন চার্ট, বার চার্ট)

4. ইউজার ম্যানেজমেন্ট টেবিল (সর্টেবল, ফিল্টারেবল, এডিটেবল)
5. সার্ভিস ম্যানেজমেন্ট ইন্টারফেস (CRUD অপারেশনস)
6. টিকেট ম্যানেজমেন্ট ইন্টারফেস (AI এবং হিউম্যান টিকেট প্রসেসিং)
```

## সার্ভিস ইন্টিগ্রেশন প্রম্পট

```prompt
1. পাঁচটি টিকেট টাইপ: রিফিল, ক্যানসেল, স্পিড আপ, রিস্টার্ট, ফেক অর্ডার কমপ্লেইন
```
